## The Experience

The application takes the user through a series of animated sections, each choreographed to feel like a directed film:

- **Video Intro** — An auto-playing brand film, shown periodically to returning users. A preload progress bar surfaces while assets load.
- **Hero** — Full-viewport product imagery with large-scale type and an animated scroll hint.
- **Canvas Sequence** — As the user scrolls, frames are painted to an HTML5 canvas — simulating cinematic video playback without a single video file.
- **Tagline Reveal** — *Timeless · Refined · Audacious · Yours* — pinned and revealed one by one.
- **Editorial** — High-res photography with parallax depth, fragrance notes, product section, and a pull-quote.
- **Navigation** — A sleek hamburger overlay, accessible at any scroll depth, fully animated.
